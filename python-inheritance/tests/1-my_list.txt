The ``MyList`` class
====================

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList([3, 1, 2])
>>> my_list.print_sorted()
[1, 2, 3]
>>> print(my_list)
[3, 1, 2]

>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.print_sorted()
[1, 4, 5]
>>> print(my_list)
[5, 1, 4]
